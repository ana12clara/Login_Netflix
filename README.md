## **Página de Login — Netflix (Clone)**

Este projeto consiste na recriação da página de login da Netflix, desenvolvida com fins educacionais.

Ele faz parte de uma série de projetos pequenos e práticos, criados para aprimorar habilidades em desenvolvimento web, especialmente na construção de interfaces modernas e responsivas.

🧠 *Objetivo do Projeto*

O principal objetivo foi reproduzir o visual da página de login, focando em:

- Estruturação correta do layout

- Estilização fiel ao design original

- Organização de código

- Atenção aos detalhes visuais

Este projeto não possui integração com backend ou autenticação real.

⚠️ Aviso Legal

Este projeto foi desenvolvido apenas para fins educacionais, sem qualquer intenção comercial.
